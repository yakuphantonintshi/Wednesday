There are different programming languages that involve web development

HTML

HTML is used to create a starting point for the website and is what most of your static pages start from

CSS

CSS is used hand in hand with HTML to add colours, backgrounds, layouts, font sizes and more.

Javascript

Javascript is used in many aspects of web development. Web developers use this language to add interactive elements to their websites.
